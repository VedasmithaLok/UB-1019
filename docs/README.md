# AI X-Ray Detection with Multilingual Report

An AI-powered chest X-ray analysis system that detects pneumonia and generates multilingual medical reports.

## Features

- AI-Powered Detection using Deep Learning
- Grad-CAM Visualization
- Multilingual Support (5 languages)
- Text-to-Speech Reports
- PDF Export & QR Code Sharing

## Quick Start

```bash
pip install -r requirements.txt
python train_fast.py
streamlit run app.py
```
